# Cafe Sales Dashboard (Microsoft Excel)

## 📌 Project Overview
This project is a **Cafe Sales Dashboard** created entirely in **Microsoft Excel** using advanced Excel tools such as Pivot Tables, Pivot Charts, Slicers, Conditional Formatting, and Dashboard Design techniques.

The dashboard provides a complete overview of cafe business performance by analyzing:
- Total Transactions
- Total Quantity Sold
- Total Revenue
- Item-wise Sales
- Monthly Sales Trends
- Day-wise Sales
- Location-wise Sales
- Payment Method Distribution
- Quantity Distribution

This dashboard helps in understanding customer purchasing behavior, product performance, revenue trends, and operational insights for better business decisions.
