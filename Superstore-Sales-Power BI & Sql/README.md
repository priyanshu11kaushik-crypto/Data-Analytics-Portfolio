# Superstore Sales & Profit Analysis Dashboard (SQL + Power BI)

## 📌 Project Overview
This project is a **Superstore Sales & Profit Analysis Dashboard** built using **SQL for data extraction, cleaning, and transformation** and **Power BI for interactive data visualization**.

The dashboard provides deep insights into:
- Sales Performance
- Profit Analysis
- Customer Segments
- Product Categories & Sub-Categories
- Regional Performance
- Yearly & Monthly Trends
- Shipping Mode Analysis

It helps businesses identify profitable areas, weak-performing categories, customer behavior, and sales trends for better decision-making.

---

## 🚀 Tools & Technologies Used
- **SQL** – Data cleaning, querying, transformation, and KPI generation  
- **Power BI** – Dashboard creation, data modeling, DAX measures, and visualization  
- **Dataset** – Superstore Sales Dataset
